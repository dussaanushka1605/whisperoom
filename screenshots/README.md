## How to Take Screenshots:

- **Windows**: Press `Win + Shift + S` to open Snipping Tool
- **Mac**: Press `Cmd + Shift + 4` to take a screenshot
- **Browser**: Use browser dev tools screenshot feature

## Tips:

- Use consistent window sizes
- Hide sensitive information if needed
- Show active features clearly
- Keep file sizes reasonable (under 1MB each)
